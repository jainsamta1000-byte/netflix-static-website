Netflix Static Website

A static Netflix user interface clone developed with HTML5 and CSS3,
showcasing strong foundational frontend skills and responsive design principles.

Features-
Clean and modern Netflix-inspired UI
Responsive design for desktop and mobile devices
Built entirely with HTML5 and CSS3 

📂 Project Structure

netflix-static-website
│── index.html        # Main HTML file
│── style.css         # Core stylesheet
│── assets/           # Images, icons, and other static files

🛠️ Setup & Usage
Clone the repository:
git clone https://github.com/jainsamta1000-byte/netflix-static-website.git

Navigate to the project folder:
cd netflix-static-website

Open index.html in your browser to view the site.

🎯 Purpose
This project is intended as a frontend practice exercise to replicate a popular streaming platform’s look and feel, focusing on layout, styling, and responsiveness
